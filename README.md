# Dooble project
<br/>
A subset demo for Dooble company.
The app was written as part of an interview assignment.

## Technologies Used
```
 React
 Material-UI
 Rick & Morty API
 Axios
 Sass
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```
